## Prerequisites

For consumption, this package is ES5 compliant and will run in most modern environments.

For development, the below are required.
```
yarn >= 1.22
node >= 12.13.0
```


## Installing

```
yarn install
```

## Testing

```
yarn test
```

## Building

```
yarn build
```
